# [Vision] 개발자를 위한 제안: 우리는 단순한 ERP가 아닌, 플랫폼을 만듭니다

## 1. 당신을 초대합니다
우리는 지금 공공과 기업의 일하는 방식을 근본적으로 혁신할 **차세대 SaaS ERP 플랫폼**을 시작하려 합니다. 
단순히 기능을 구현하는 코더(Coder)가 아닌, 복잡한 비즈니스 문제를 기술로 해결하고 시스템의 아키텍처를 주도할 **엔지니어(Engineer)**로서 당신을 초대합니다.

## 2. 우리가 마주할 기술적 도전 (Technical Challenges)
이 프로젝트는 단순한 CRUD 애플리케이션이 아닙니다. 당신은 다음과 같은 흥미로운 도전들을 마주하게 될 것입니다.

- **MSA & Distributed System**: 모놀리식의 한계를 넘어, 수십 개의 마이크로서비스가 유기적으로 동작하는 분산 시스템을 설계하고 운영합니다.
- **AI & LLM Integration**: 단순한 API 연동을 넘어, RAG(Retrieval-Augmented Generation) 파이프라인 구축, 프롬프트 엔지니어링, Vector DB 활용 등 최신 AI 기술을 프로덕션 레벨에서 적용합니다.
- **Multi-tenancy Architecture**: 하나의 인스턴스로 수많은 고객사를 지원하면서도, 완벽한 데이터 격리와 커스터마이징을 제공하는 고난도 아키텍처를 구현합니다.
- **High Performance & Scalability**: 대용량 트래픽과 데이터를 효율적으로 처리하기 위한 캐싱, 큐잉, 샤딩 등 다양한 최적화 기법을 적용합니다.
- **Modern DevOps**: Kubernetes, Docker, CI/CD 파이프라인을 통해 코드가 프로덕션에 배포되는 전 과정을 자동화하고 제어합니다.

## 3. 당신이 얻게 될 성장 (Growth Opportunities)
이 프로젝트에 합류한다는 것은 시장에서 가장 수요가 높은 기술과 경험을 내재화한다는 것을 의미합니다.

- **Full-stack Mastery**: NestJS(Backend)와 Next.js(Frontend)라는 현재 가장 강력한 조합을 깊이 있게 다루며 풀스택 역량을 완성합니다.
- **AI Engineering**: LLM을 활용한 실제 서비스를 구축해보며, 미래 개발자의 필수 역량인 AI 엔지니어링 스킬을 확보합니다.
- **System Design**: 대규모 트래픽을 감당하는 시스템 설계 능력을 갖추게 됩니다.
- **Domain Expertise**: HR, 재무 등 엔터프라이즈 도메인 지식을 습득하여 대체 불가능한 전문가로 성장합니다.

## 4. 우리의 개발 문화 (Engineering Culture)
- **주도적 설계**: "시키면 하는" 개발이 아닙니다. 기획 단계부터 참여하여 기술적 관점에서 더 나은 해결책을 제안하고 결정합니다.
- **코드 품질에 대한 집착**: 기능 동작은 기본입니다. 우리는 읽기 쉽고, 테스트 가능하며, 유지보수하기 쉬운 코드를 작성하는 것을 타협하지 않습니다.
- **상호 존중과 성장**: 코드 리뷰는 비난이 아닌 성장의 기회입니다. 서로의 지식을 공유하고 함께 성장하는 문화를 지향합니다.
