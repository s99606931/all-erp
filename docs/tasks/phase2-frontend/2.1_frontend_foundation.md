# Task 2.1: Frontend Design System Foundation

## 1. 개요 (Overview)

- **Task ID**: 2.1
- **목표**: 11개 MFE 앱에서 공통으로 사용할 **디자인 시스템 라이브러리(`libs/shared/ui-design`)**를 구축하고, **Tailwind CSS v4**와 **Shadcn/UI**를 초기화한다.
- **선행 작업**: 없음

## 2. 상세 요구사항 (Requirements)

1.  **Shared Library 생성**: `libs/shared/ui-design` 경로에 React 라이브러리 생성.
2.  **Tailwind CSS v4 설정**:
    - Zero-runtime 엔진 적용.
    - Global CSS Variables (Token) 정의 (`:root { --primary: ... }`).
3.  **Shadcn/UI 초기화**:
    - `cn()` 유틸리티 함수 구현.
    - 기본 컴포넌트(Button, Input, Card, Dialog) 설치.
4.  **폰트 설정**: `Inter` (기본) 및 `JetBrains Mono` (코드) 폰트 적용.

## 3. 기술적 의사결정 (Technical Decisions)

- **Tailwind Version**: **v4** (Alpha/Beta 버전인 경우 v3.4 최신 안정 버전 사용하되, v4 문법 지향). _안정성을 위해 v3.4 우선 적용 후 v4 마이그레이션 고려 (User Confirmation 필요)._ -> **결정: v3.4 (안정성 우선)**
- **Colors**: Slate (Gray), Indigo (Primary), Rose (Destructive) 테마.

## 4. 작업 단위 (Work Units)

1.  **라이브러리 생성**:
    ```bash
    pnpm nx g @nx/react:lib ui-design --directory=libs/shared/ui-design --style=css --linter=eslint --unitTestRunner=jest
    ```
2.  **의존성 설치**:
    ```bash
    pnpm add -w tailwindcss postcss autoprefixer class-variance-authority clsx tailwind-merge lucide-react
    ```
3.  **Tailwind 설정**:
    - `libs/shared/ui-design/tailwind.config.ts` 생성.
    - `libs/shared/ui-design/src/lib/styles/global.css` 생성 및 기본 테마 정의.
4.  **Shadcn/UI 유틸리티**:
    - `libs/shared/ui-design/src/lib/utils.ts`에 `cn` 함수 작성.
    - `components.json` 설정 파일 생성 (루트 또는 라이브러리 내부).

## 5. 승인 기준 (Acceptance Criteria)

- [ ] `libs/shared/ui-design` 라이브러리가 생성되어야 한다.
- [ ] `Button` 컴포넌트를 import하여 Storybook 또는 샘플 페이지에서 렌더링했을 때 스타일이 적용되어야 한다.
- [ ] Tailwind 클래스(`bg-primary`)가 정상적으로 적용되어야 한다.
